# Student Performance Data Analysis

## Overview
This project analyzes student exam performance using Pandas and Matplotlib.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Analysis
- Average score calculation
- Bar chart by gender
- Scatter plot analysis
- Correlation heatmap

## Conclusion
The analysis highlights performance trends and relationships between exam scores.



# House Price Prediction using Linear Regression

## Objective
To predict house prices based on features such as number of rooms, area, and location.

## Dataset
Collected from Kaggle – housing price dataset.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Model Used
- Linear Regression

## Evaluation Metrics
- MAE
- RMSE
- R² Score

## Conclusion
The linear regression model effectively predicts house prices and highlights key influencing factors.
